# Walchand College of Engineering Website

This project is a simple static website created for Walchand College of Engineering, Sangli.  
It demonstrates a basic college website layout with navigation, search functionality, and a chatbot assistant.

## Features

- Responsive homepage layout
- Navigation menu for different sections
- Search bar to quickly find pages
- Chatbot assistant for basic navigation
- News ticker section
- Cards for Academics, Placements, and Departments

## Technologies Used

- HTML
- CSS
- JavaScript
- Font Awesome

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.

## Hosting

This website can be hosted using GitHub Pages.

Steps:
1. Upload the project to a GitHub repository.
2. Go to repository Settings.
3. Open Pages settings.
4. Select the main branch as the source.
5. Save and wait for the deployment.

The website will be available at:

https://username.github.io/repository-name
